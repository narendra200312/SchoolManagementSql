
# Student Course Management - SQL Project

This project is a simple relational database for managing students, courses, and enrollments using SQL.

## Files

- `schema.sql` – Creates database and tables
- `data.sql` – Inserts sample data
- `queries.sql` – Example SQL queries

## How to Run

```sql
-- Run schema
SOURCE schema.sql;

-- Insert data
SOURCE data.sql;

-- Run example queries
SOURCE queries.sql;
